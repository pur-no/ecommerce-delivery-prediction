E-commerce Delivery Prediction

Predicting on-time shipment delivery using Machine Learning

About the Project

This project aims to predict whether an e-commerce shipment will reach the customer on time based on shipment, product, and customer-related features. The project was developed as part of the CSE422: Artificial Intelligence course and applies both supervised and unsupervised learning techniques.

Dataset Overview

  • Total Samples: 10,999
  • Input Features: 11
  • Target Variable: Reached.on.Time_Y.N (Binary)
  • Feature Types: Numerical & Categorical

Models Used

Logistic Regression
K-Nearest Neighbors (KNN)
Decision Tree
Neural Network
K-Means Clustering (Unsupervised)

Results & Insights

All supervised models achieved comparable accuracy; however, due to dataset imbalance, evaluation was performed using precision, recall, F1-score, and ROC–AUC. The Neural Network demonstrated the most balanced overall performance across these metrics.

  • Best Model: Neural Network
  • Accuracy: ~68%
  • ROC–AUC: > 0.7
